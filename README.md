# CMD Guide — Guia Completo do Prompt de Comando do Windows

  Guia completo e interativo do CMD (Prompt de Comando do Windows) em Português Brasileiro, com 29 páginas cobrindo do básico ao avançado.

  ## 🌐 Demo Online

  **[Acesse o guia em: wallysondevs.github.io/cmd-guide](https://wallysondevs.github.io/cmd-guide/)**

  ## 📋 Conteúdo

  - **Introdução**: História do CMD, Primeiros Passos
  - **Fundamentos**: Ajuda, Variáveis, Operadores, Caracteres Especiais
  - **Sistema de Arquivos**: Navegação, Arquivos, Conteúdo, Atributos
  - **Texto e Dados**: Strings, Filtros, Redirecionamento
  - **Scripts**: Criando Scripts, Fluxo de Controle (IF/GOTO), Loops (FOR), Funções, Erros
  - **Avançado**: Processos, Serviços, Rede, Usuários, Permissões, Registro, WMIC, Agendamento
  - **Referência**: Dicas e Truques, Referência Rápida

  ## 🛠️ Tecnologias

  - **React** + **TypeScript** + **Vite**
  - **Tailwind CSS** com tema verde terminal
  - **Wouter** (roteamento hash-based)
  - **Framer Motion** (animações)
  - **React Syntax Highlighter** (highlight de código)

  ## 🚀 Rodando Localmente

  ```bash
  npm install
  npm run dev
  ```

  Acesse http://localhost:3000

  ## 🏗️ Build

  ```bash
  BASE_PATH=/ PORT=3000 npm run build
  ```
